Guide

The Capstone Project Library Mobile App is designed to be user-friendly and intuitive. Here are some of the features and functions of the app:

Home Tab

The home tab displays a list of featured capstone projects, such as stem and abm.

Search Tab

The search tab has a search bar located at the top of the screen for finding projects based on keywords or title

Add Button

This function saved capstone projects for the user and displays them in the "Library" tab of the app.

Read Button

This function will open the user's default app for viewing capstone project files.
